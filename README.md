# Internship Tracker Web App

## Description
The Internship Tracker Web App is a web-based application designed to help students manage and track their internship applications efficiently. Users can add internship details, update application status, and keep track of their progress throughout the internship application process.

## Technologies Used
- HTML
- CSS
- Python (Flask)
- MySQL

## Features
- Add internship applications
- Update application status (Applied, Interview, Selected, Rejected)
- Delete internship entries
- View and manage application records

## Future Improvements
- Dashboard with analytics and statistics
- Email reminders for follow-ups
- User authentication system (login/signup)
